Currently it's supported 5 field types, they are:

=====================================================================
Type	|    Meaning			|   Metadata		    |
=====================================================================
Byte 	| A byte			| ~			    |
---------------------------------------------------------------------
Int16	| 2 Bytes signed integer number	| ~			    |
---------------------------------------------------------------------
Int32	| 4 Bytes signed integer number	| ~			    |
---------------------------------------------------------------------
Int64	| 8 Bytes signed integer number	| ~			    |
---------------------------------------------------------------------
String	| A String			| size (Can be a command)   |
---------------------------------------------------------------------
